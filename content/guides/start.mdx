# 📟 Терминал

Терминал — новейшая разработка компании MultiPUNK. Устройства для мобильного управление игровым контентом

<Callout type="info">
Вы потеряли свой терминал? Не проблема, мы подготовили для вас новый терминал на спавне!
</Callout>

### Терминал открывает доступ к:

#### Достижениям
- Выполняйте задания в категориях.
- За каждое выполненное испытание вы получаете **самоцветы** — валюту сервера.

#### Заданиям
- Задания обновляются **ежедневно**.
- За выполнение выдаются **дополнительные самоцветы**.
- Получайте награды **до обновления списка** — они не копятся.

#### Магазинам
- Магазин обновляются **ежедневно**.
- Позволяет покупать уникальные предметы

#### Таблица лидеров (Система FLow)
- Список лидеров по количеству выполнених достижений
