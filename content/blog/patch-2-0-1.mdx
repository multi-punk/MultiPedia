# ⚙️ Небольшое обновление 2.0.1

![Небольшое обновление 2.0.1](/assets/news/new_graphic_smp_update_no_picture.png)



## 📝 Список изменений и нововведений:

- Добавлен способ создания увеличительного стекла который вы сможете использовать для как декоративный предмет а обладатели подписки начиная от "Создвездие" смогут с помощью неё проверять логи взаимодействий.

![Крафт увеличительного стекла](/assets/news/lupa.png)

---

## ⚙️ Исправлены баги:

Исправлены нерабочие достижения:
- Исправлено достижение "Точно в цель"
- Исправлены достижения "Это птица?", "Это воздушный шар?", "Это самолёт?"
- Исправлено достижение "Играй, гармония!"
- Исправлено достижение "Толстосум"
- Исправлено достижение "Проверенно временем"

Прочие исправления:
- Исправлена проблема при которой создание субрегионов при изменение границ территории не представлялось возможным
- Теперь во всех рецептах готовки в которых использовались яйца можно использовать любой тип яиц
- Исправлен баг при котором предметы могли удалятся с арен боссов
- Исправлен баг при котором координаты могли не отоброжаться с компасом в инвентаре
- Теперь ящик с луком можно разделить на 9 лука, также как и другие ящики
